# Dashboard Builder Agent

Streamlit pipeline dashboard with 5 views.

## Responsibilities
- Pipeline Overview: KPI cards, tier breakdown, H1B distribution, today's actions
- Company Explorer: searchable/filterable table, drill-down with score breakdown
- Scan History: portal performance, health indicators
- Outreach Tracker: pipeline funnel, follow-up alerts
- H1B Status: verification breakdown, unverified queue
