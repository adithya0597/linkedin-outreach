# Networking Log (v2.0 — 2026 Optimized)

Track all outreach activity. Update after every interaction.

---

## Status Code Reference

| Code | Meaning | Next Action | Timeline |
|------|---------|-------------|----------|
| Researched | Intel gathered, no outreach yet | Begin pre-engagement (Template 0) | Day 0 |
| Post Engaged | Commented on their LinkedIn post | Send connection request (Template 1) | Day 1 |
| Connection Sent | Request sent, awaiting acceptance | Wait 5-7 days | 5-7 days |
| Connected | Connection accepted | Send follow-up within 2 days (Template 2) | Immediate |
| Follow-Up Sent | First follow-up sent | Wait 4-5 days for response | 4-5 days |
| In Conversation | 2+ messages exchanged | Continue dialog or move to application | Varies |
| Application Submitted | Applied to the role | Follow up with contact after 7 days (Template 6) | 7 days |
| Interview Scheduled | Interview confirmed | Run interview prep (Prompt 9) | Before interview |
| Interview Completed | Interview round finished | Send thank-you within 24 hours | 24 hours |
| Offer Received | Formal offer extended | Negotiate or accept/decline | 1-2 weeks |
| Awaiting Response | Message sent, waiting on reply | Re-engage after 4-5 days (Template 3) | 4-5 days |
| Stale (No Response) | No response after 3+ touches | Archive or deprioritize | Move to archive |
| Declined (Self) | You declined opportunity | Send gracious goodbye (Template 7A) | 24 hours |
| Rejected | They rejected you | Send Template 7B if interview stage; otherwise archive | 24 hours |
| Deprioritized | Low priority, removed from active | No further action unless reactivated | N/A |

## Enhanced Column Definitions

| Column | Purpose | Example |
|--------|---------|---------|
| Date | When action was taken | 2026-03-05 |
| Name | Full name of contact | Sarah Chen |

| Action | What you did | Sent connection request (Template 1B) |
| Status | Current status (see codes above) | Connection Sent |

| Message Type | Which template/variant | Template 1B (Professional) |
| Response Time (days) | Days from send to response | 3 |
| Fit Score (1-5) | Role fit assessment | 4 |
| Days Since Last Action | Auto-calculate or manual update | 5 |
| Output Folder | Link to swarm output | /Databricks_SeniorML/ |
| Next Touch | Planned follow-up date + action | 2026-03-09: Follow-up (Template 2B) |
| Notes | Additional context | Mentioned RAG in recent post; high priority |
