## CRITICAL RULES (Do NOT Break)

### ✅ Green Flags (MUST Have 5+)
- Message references specific detail about recipient (their role, post, company)
- Includes 1 quantified stat (role-specific)
- Natural tone (not templated)
- No generic phrases
- Character count verified and within limit
- Portfolio URL in correct place (follow-ups only, NEVER connection request)
- No job ask in first contact
- Tone matches recipient
- Sentence structure varies from previous messages
- Clear next step included

### ❌ Red Flags (Do NOT Send)
- Starts generic without personalization
- Asks for job in first message
- Portfolio in connection request
- Multiple stats crammed together
- Same structure as previous message
- Generic reference to company
- Tone mismatch
- Typos or grammar errors
- Exceeds character limit
- Person already contacted recently
