# DECISION TREE: What To Do If...

## Scenario 1: Connection Accepted + Replying (✅ BEST CASE)
```
→ Aisha/Boris replied to your follow-up message
→ They're asking questions or showing interest
→ Action: Respond within 24 hours, match their tone
→ Next: Move toward phone call ("30-min chat about [topic]?")
→ Timeline: Phone call could happen within 5-7 days
```

## Scenario 2: Connection Accepted + No Reply (⚠️ NORMAL)
```
→ They accepted but haven't replied to follow-up (5+ days silent)
→ Action: Send re-engagement message (Template 3) with fresh angle
→ Next: If still no reply after 3rd touch, archive and move to backup contacts
→ Timeline: 14 days max per contact, then escalate
```

## Scenario 3: Connection Not Accepted (❌ EXPECTED — Some Won't Accept)
```
→ 7+ days, still "Pending"
→ Action: Skip the follow-up, move directly to other contacts or different outreach angle
→ Next: Try contacting via InMail (if credits available) or backup recruiter
→ Timeline: Max 2 weeks of waiting, then escalate or deprioritize
```

## Scenario 4: Application Acknowledged, No Recruiter Response
```
→ Greenhouse shows "submitted" but no message from recruiter after 7 days
→ Action: Post-application follow-up message (Template 6) to hiring team
→ Next: Interview scheduling, technical screen, or rejection (takes 1-2 weeks typically)
→ Timeline: Allow 14 days for application review cycle
```

## Scenario 5: Interview Scheduled (🎯 MOVING FORWARD)
```
→ Congratulations! You have an interview.
→ Action: Shift focus to interview prep (COWORK_CHEATSHEET.md Prompt 9)
→ Next: Prepare STAR stories, tech questions, research interviewer
→ Timeline: 3-5 days of prep before interview
```

## Scenario 6: CEO InMail No Reply (Expected — Low Priority for Execs)
```
→ Vipul Ved Prakash didn't reply to your InMail
```
